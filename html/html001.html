(1) h태그에 대해서 설명 : h태그는 margin이 존재하여 여백이 있습니다.

제일 글씨큰 헤더 32px : 맑은 고딕체

두번째로 글씨큰 헤더 24px : 맑은 고딕체

세번째로 글씨큰 헤더 약19px : 맑은 고딕체

네번째로 글씨큰 헤더 16px : 맑은 고딕체

디섯번째로 글씨큰 헤더 약13px : 맑은 고딕체
여섯번째로 글씨큰 헤더 약11px : 맑은 고딕체

(2) img, br, hr 태그 등은 종료 태그 없이 시작 태그만을 가지는데 이러한 태그들을 빈 태그(empty tag)라고 합니다.
